Project Description

This project is a proxy server that is meant to download any available file from a url. It allows for the options of choosing the conditions of the download. The client and server programs allow for command line function to select whether you want to use the stop-and-wait protocol or the sliding window protocol. It also allows for us of ipv4 or ipv6 addresses, and the simulation of dropped packets.
